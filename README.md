# Background Removal and Image Cropping Tool

A Python-based tool to remove backgrounds from images and extract non-transparent pixels with cropping.

## Features

- **Background Removal**: Uses `rembg` to remove backgrounds from images
- **Smart Cropping**: Automatically crops to non-transparent/non-black pixels
- **Base64 Support**: Handles Base64 encoded images
- **Flexible Output**: Saves processed images in PNG format
